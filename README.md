# data_cleaning_visualization
A Data Cleaning and Visualization project using Python, Pandas, Matplotlib, and Seaborn. The project includes handling missing values, removing duplicates, detecting outliers, and creating visualizations to derive meaningful insights from employee data through exploratory data analysis.
# Data Cleaning and Visualization

## Project Overview

This project demonstrates the process of data cleaning and visualization using Python. The Student Performance dataset is analyzed to identify missing values, remove duplicate records, generate summary statistics, and create visual representations of the data.

## Objectives

* Load and inspect the dataset
* Check for missing values
* Identify and remove duplicate records
* Save the cleaned dataset
* Generate statistical insights
* Create visualizations using charts

## Tools and Technologies

* Python
* Pandas
* Matplotlib
* Google Colab
* GitHub

## Files Included

* StudentsPerformance.csv – Original dataset
* data_cleaning.py – Python script
* cleaned_students.csv – Cleaned dataset
* gender_chart.png – Gender distribution chart
* score_histogram.png – Math score distribution histogram
* Data_Cleaning_Visualization.ipynb – Google Colab notebook
